Scrapy中Xpath学习小结

因为最近要用Scrapy从网上抓取数据，重温了Xpath的用法。Scrapy的selector是实现的。

写这篇博客之前，先说明下我现在的写作日期是2014年3月19日，Scrapy的版本是0.22.2。因为Scrapy的版本关系，有些指令已经有变化。比如原来的指令hxs现在已经改为了sel。网上好多的教程贴都存在类似问题，有时指令都已经不好用了，还是建议啃官方的manual。

有关Xpath的语句，我感觉常用的就是那么几种。掌握了基本思想之后感觉和SQL差不多。下面的参考链接给了比较全。

这里介绍了在scrapy shell中进行测试。

http://doc.scrapy.org/en/latest/_static/selectors-sample1.html

测试网页就是这个。

<html>
 <head>
  <base href='http://example.com/' />
  <title>Example website</title>
 </head>
 <body>
  <div id='images'>
   <a href='image1.html'>Name: My image 1 <br /><img src='image1_thumb.jpg' /></a>
   <a href='image2.html'>Name: My image 2 <br /><img src='image2_thumb.jpg' /></a>
   <a href='image3.html'>Name: My image 3 <br /><img src='image3_thumb.jpg' /></a>
   <a href='image4.html'>Name: My image 4 <br /><img src='image4_thumb.jpg' /></a>
   <a href='image5.html'>Name: My image 5 <br /><img src='image5_thumb.jpg' /></a>
  </div>
 </body>
</html>

首先打开shell，建议安装ipython。

scrapy shell http://doc.scrapy.org/en/latest/_static/selectors-sample1.html

一般的selector

当加载shell之后，sel就可以用了。

>>> sel.xpath('//title/text()')
[<Selector (text) xpath=//title/text()>]

一般是.extract()讲内容取出。

下面也介绍了一些

>>> sel.xpath('//title/text()').extract()
[u'Example website']

>>> sel.xpath('//base/@href').extract()
[u'http://example.com/']

>>> sel.css('base::attr(href)').extract()
[u'http://example.com/']

>>> sel.xpath('//a[contains(@href, "image")]/@href').extract()
[u'image1.html',
 u'image2.html',
 u'image3.html',
 u'image4.html',
 u'image5.html']

>>> sel.css('a[href*=image]::attr(href)').extract()
[u'image1.html',
 u'image2.html',
 u'image3.html',
 u'image4.html',
 u'image5.html']

>>> sel.xpath('//a[contains(@href, "image")]/img/@src').extract()
[u'image1_thumb.jpg',
 u'image2_thumb.jpg',
 u'image3_thumb.jpg',
 u'image4_thumb.jpg',
 u'image5_thumb.jpg']

>>> sel.css('a[href*=image] img::attr(src)').extract()
[u'image1_thumb.jpg',
 u'image2_thumb.jpg',
 u'image3_thumb.jpg',
 u'image4_thumb.jpg',
 u'image5_thumb.jpg']

有关根据css选取的我还不太会，觉得也不太直观。这一方面还在学习当中。

使用正则表达式

还有一点比较方便的是使用正则表达式来选取。

即用.re 代替.extract()

>>> sel.xpath('//a[contains(@href, "image")]/text()').re(r'Name:\s*(.*)')
[u'My image 1',
 u'My image 2',
 u'My image 3',
 u'My image 4',
 u'My image 5']

想对位置的Xpath

>>> divs = sel.xpath('//div')
>>> for p in divs.xpath('//p')  # this is wrong - gets all <p> from the whole document
>>>     print p.extract()

正确的做法应该是下面的

>>> for p in divs.xpath('.//p')  # extracts all <p> inside
>>>     print p.extract()

或者是这种

>>> for p in divs.xpath('p')
>>>     print p.extract()
