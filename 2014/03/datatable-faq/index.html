data.table-案例学习-按条件筛选

我们来看一个简单的问题， 下面的数据框有三个字段。SIC，FYEAR，和BIG4.

任务: 找出每个SIC，Fyear大于7年的记录。然后将其他的删除。

     SIC FYEAR BIG4
  1:   1  1999    0
  2:   1  1999    0
  3:   1  1999    1
  4:   1  1999    1
  5:   1  1999    1
  6:   1  2000    0
  7:   1  2000    0
  8:   1  2000    0
  9:   1  2000    1
 10:   1  2000    1
 11:   1  2000    1
 12:   1  2000    1
 13:   1  2000    1
 14:   1  2000    1
 15:   1  2000    1
 16:   1  2001    0
 17:   1  2001    0
 18:   1  2001    0
 19:   1  2001    1
 20:   1  2001    1

实现这一目的并不难，对于不同的包深圳相同的包都有不同的方法。这里展示几个例子

1： plyr

newdata <- ddply(data, .(SIC,FYEAR), function(x){if(nrow(x)>7) x else NULL});

2：data.table

#1 using .SD
##1.1
foo.SD <- function(x, .by,.thresh){
        x[,if(.N > .thresh){.SD},by=.by]
      }
##1.2
foo.SD <- function(x, .by,.thresh){
        x[,.SD[.N > .thresh], by = .by]
      }
## 注意1.1和1.2思想是非常相似的，在实现的时候有一点点差别。大家要仔细体会下

#2 using .I 这个理解起来稍微有点绕，但是算起来比.SD的方法要快一点
foo.I <- function(x, .by,.thresh){
       x[x[,if(.N>.thresh){.I},by=.by]$V1]
 }

foo.SD(data, c('SIC','FYEAR'), 7)
foo.I(data, c('SIC','FYEAR'), 7)
