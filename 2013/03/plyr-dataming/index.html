Plyr 拓展包快速入门

大神Hadley Wickham写的Plyr拓展包，是所有拓展包里下载最多的。联合上Reshape2，和GGplot2，已经成为业界口碑最好的。

今天着重介绍DDPly()函数

  • transform(df, var1 = expr1, ...)
  • summarise(df, var1 = expr1, ...)

注意两者差别

  • Transform modiﬁes an existing data frame.
  • Summarise creates a new data frame.
